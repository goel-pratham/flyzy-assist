# FlyZy Assist – Travel Service Chatbot

FlyZy Assist is a beginner-friendly Python travel chatbot developed during the 10-week Virtual Global Apprenticeship in Artificial Intelligence with FlyZy through Belong Education.

The chatbot is designed to help users with common travel and hotel-related queries such as:
- hotel check-ins and check-outs
- cab bookings
- flight-related questions
- food and restaurant services
- WiFi access
- payment methods
- hotel location assistance

This project was built using Python and basic Natural Language Processing (NLP) concepts such as intent matching, tokenization, and conversational memory handling.

# How the Chatbot Works

The chatbot follows a simple rule-based conversational flow:

1. Takes user input
2. Cleans punctuation from the sentence
3. Tokenizes the sentence into words
4. Detects user intent using keyword matching
5. Generates an appropriate response
6. Stores short-term conversational memory for follow-up questions

The chatbot can also remember the user’s previous request for limited contextual conversation.
